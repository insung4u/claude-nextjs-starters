# 프로덕션 빌드

프로덕션 빌드를 수행하고 결과를 확인합니다.

## 실행 순서

1. **빌드 실행**
```bash
npm run build
```

2. **빌드 결과 확인**
- `.next/` 폴더 생성 확인
- 빌드 로그에서 에러 또는 경고 확인
- 페이지별 번들 크기 확인

3. **프로덕션 서버 테스트 (선택)**
```bash
npm run start
```

## 체크리스트

- [ ] TypeScript 타입 에러 없음
- [ ] 빌드 경고 확인 및 필요시 수정
- [ ] 정적 페이지 생성 성공
- [ ] 이미지 최적화 적용 확인

## 일반적인 빌드 에러

1. **타입 에러**: `tsconfig.json`의 strict 모드로 인한 타입 불일치
2. **Import 에러**: 경로 별칭 `@/` 확인
3. **서버/클라이언트 혼동**: `"use client"` 지시문 확인
