# 전체 코드 검증

린트와 빌드를 모두 실행하여 코드 품질을 종합적으로 검증합니다.

## 실행 순서

1. **린트 검사**
```bash
npm run lint
```

2. **빌드 검증**
```bash
npm run build
```

## 검증 체크리스트

### 린트 (ESLint)
- [ ] 린트 에러 없음
- [ ] 린트 경고 확인 및 검토

### 타입 검사 (TypeScript)
- [ ] 타입 에러 없음
- [ ] strict 모드 준수

### 빌드 (Next.js)
- [ ] 빌드 성공
- [ ] 번들 크기 적정
- [ ] 정적 페이지 생성 완료

## 문제 해결 순서

1. 린트 에러 먼저 수정
2. 타입 에러 수정
3. 빌드 에러 수정
4. 경고 사항 검토 및 선택적 수정

## CI/CD 파이프라인 명령어

```bash
npm run lint && npm run build
```
